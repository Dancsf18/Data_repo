📌 Descripción

Este proyecto implementa un modelo de clasificación supervisada para determinar si una solicitud de tarjeta de crédito será aceptada (1) o rechazada (0) a partir de variables demográficas y financieras.

⚙️ Contenido del notebook

Carga y exploración del dataset credito.txt.

Limpieza y preprocesamiento de datos.

Entrenamiento y comparación entre Regresión Logística y Random Forest.

Evaluación de desempeño mediante métricas (Accuracy, Recall, Precision, F1 y AUC).

Curvas ROC, matriz de confusión y conclusiones del modelo.

📊 Resultados

Mejor modelo: Random Forest (AUC = 0.956).

Muestra buena capacidad predictiva, estable y sin sobreajuste.

Se interpreta como un clasificador robusto y bien calibrado.

💡 Tecnologías

Python · Scikit-learn · Pandas · Matplotlib · Seaborn
